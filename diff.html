The Differences between XR and other Virtual Simulation technologies

As eXtended Reality combines the features of AR, MR, and VR technologies into one single technology term, they are almost indistinguishable between the differring types of virtual simulation technologies that are currently in use. However, there are still key differences between the three technologies in many terms other than being on-par with these types of technologies.

Differences between XR and AR

Here is a table listing the various features that are in XR that are not present in AR.

Virtual Simulation Features
eXtended Reality Usually the technology is embedded and not completely separate
The features of VR is decided by the Virtuallity Spectrum
Augmented Reality Uses cameras and dedicated AR software
Mixes in virtual objects with real-world enviroments
Virtual objects are not manipulated by special gloves and/or headset

Differences between XR and MR

Here is a table listing the various features that are in XR that are not present in MR.

Virtual Simulation Features
eXtended Reality Usually the technology is embedded and not completely separate
The features of VR is decided by the Virtuallity Spectrum
Mixed Reality Uses a dedicated headset desinged for MR
Mixes in virtual objects with real-world enviroments like with AR
Allows people to manipulate virtual objects as if they're real (Requires a headset and special gloves)

Differences between XR and VR

Here is a table listing the various features that are in XR that are not present in VR.

Virtual Simulation Features
eXtended Reality Usually the technology is embedded and not completely separate
The features of VR is decided by the Virtuallity Spectrum
Virtual Reality Uses a standalone headset or a VR accessory for a smartphone
Enviroment is purely virtual and computer-generated
Most popular virtual simulation technology along with others
